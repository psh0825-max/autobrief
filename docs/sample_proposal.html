박서준 대표님 안녕하세요,

문의주신 소품샵 재고 관리 웹앱에 대한 제안을 드립니다. 저희는 대표님께서 겪고 계신 수동 재고 관리의 어려움을 깊이 공감하며, 직원들이 더 효율적으로 일할 수 있는 직관적이고 안정적인 맞춤형 솔루션을 제공하고자 합니다.

우리의 접근 방식

검증된 Brief → Prototype → Launch → Improve 4단계 프로세스로 아이디어를 현실로 만듭니다.

1
Brief & Design (1주차)
킥오프 미팅으로 요구사항을 명확히 하고, UX 중심의 화면 설계(와이어프레임)를 진행합니다.
2
Prototype & Build (2–4주차)
실제 작동하는 웹앱 개발에 착수하고, 매주 진행 상황을 공유하며 피드백을 즉시 반영합니다.
3
Launch & Test (5주차)
완성된 앱을 배포하고, 대표님과 직원분들이 직접 사용하며 최종 피드백을 받는 집중 테스트를 진행합니다.
4
Improve & Handover
피드백을 반영해 안정성을 높인 뒤, 전체 소스 코드와 운영 권한을 이전합니다.

제안 범위

기본 CRUD SaaS MVP (인증 · 데이터 · 대시보드)

  • 직원별 계정 생성 및 로그인
  • 상품 정보(이름·가격·수량 등) 등록 / 수정 / 삭제
  • 실시간 재고 현황을 시각적으로 보여주는 대시보드

인증 + 결제 (Stripe)

  • Stripe 연동으로 월 구독료를 정기 결제받는 시스템
